==========================
SciPy 1.6.1 Release Notes
==========================

.. contents::

SciPy 1.6.1 is a bug-fix release with no new features
compared to 1.6.0.

Please note that for SciPy wheels to correctly install with Pip on
macOS 11, Pip >= 20.3.3 is needed.
