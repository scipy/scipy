==========================
SciPy 1.13.1 Release Notes
==========================

.. contents::

SciPy 1.13.1 is a bug-fix release with no new features
compared to 1.13.0. The version of OpenBLAS shipped with
the PyPI binaries has been increased to 0.3.27.
