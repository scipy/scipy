Introduction of NumPy
----------------------

SciPy builds on NumPy, and for all basic array handling needs you can
use NumPy functions:

    >>> import numpy as np
    >>> np.some_function()

The details of NumPy functions are available in the
`Numpy Reference Guide <https://numpy.org/devdocs/reference/index.html>`_
or
`Numpy Tutorials <https://numpy.org/devdocs/user/quickstart.html>`_
or by using the :func:`help`, :func:`info` and :func:`source` commands.
